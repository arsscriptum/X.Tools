##  x.processkill

	Usage: xpk.exe [-h][-v][-n][-x] [ process name or id ]
	   -v          Verbose mode
	   -h          Help
	   -s          Search mode: kill process with name matching   DANGER *
	   -w          WhatIf: do not actually kill process.   Suggested use with -s.
	   -n          No banner
	   -x code     Exit Code
